# Project03
A clean, responsive utility website designed for the SW LBU team. The interface allows users to cross-reference enamels, primers, exterior, and interior paint tiers from major competitive brands against equivalent product alternatives found at Lowe's (such as Valspar and HGTV Home by Sherwin-Williams), complete with a visual grid framework.
