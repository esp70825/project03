# Project Retrospective
## 1. What Went Well?
The site successfully met my goals. I really wanted this project to be well put together so that I could share it with my manager. Anticipating potential issues early on helped me stay mindful of how to manage my time and ensure the project met my expectations. The work breakdown helped me visualize all the work that would be neccessary to complete this project, eliminating my procrastination risk.  

---

## 2. What Went Wrong?
Nothing went drastically wrong, but the CSS was very time consuming and required a lot of trial and error. Resources like W3Schools and YouTube CSS tips and tricks videos were extremely helpful when I encountered issues or needed to learn how to do new things such as making a table, adding images, etc. My main issue was that I could not figure out how to have the table render correctly on a mobile device, it may be because it was just a big element.

---

## 3. What Would You Do Differently?
I would add product names under each of the images to implement a search functionality to find specific products faster. To be able to add more products, I would probably make different pages to help with organization and not visually overwhelm the user.
