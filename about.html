FSA Karen Espino @1771

Hello, this is Karen Espino, FSA over at 1771 in Cockrell Hill. This site was a project for my computing applications class and I decided that I could build a site that could be useful to our team. Looking for a product crossover sheet in the store or looking for the pictures we took of them when we started can take some time to find, so I decided to make a web version that we can quickly pull up! This was very fun to make, and will hopefully come in handy for everyone!

Methodology

This website was developed to simplify product comparison. By mapping major competitive paint lines directly to equivalent products available at Lowe's this application allows users to quickly match performance tiers without the guesswork. The comparison methodology was provided by Lowe's. The site was built as a final project to showcase a clean and accessbile web design as well as data presentation.
